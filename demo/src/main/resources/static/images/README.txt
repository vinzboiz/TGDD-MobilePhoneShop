Place your storefront banner images in this folder.

Expected filenames (you can change them in templates if needed):
- top-banner.jpg           -> Top promo strip at the very top
- hero-main.jpg            -> Large center hero banner under the navbar
- hero-side-left.jpg       -> Left sticky side banner
- hero-side-right.jpg      -> Right sticky side banner

You can replace these with any JPG/PNG images sized similarly to the
original site (for example 1920x60 for the top banner and around
950x360 / 160x500 for the hero banners).
