Icon menu theo danh mục cha (parent category):
- Mục đầu tiên trên menu: dùng file /images/Dienthoaiicon.png (đã cấu hình sẵn).
- Mục thứ 2, 3, 4...: đặt file icon tại đây với tên icon-{id}.png
  Ví dụ: icon-2.png, icon-3.png, icon-4.png (id = id của ParentCategory trong DB).
Mỗi mục menu sẽ hiển thị đúng icon tương ứng.
